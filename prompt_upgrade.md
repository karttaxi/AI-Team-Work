# 로그라이크 자동 슈팅 게임 개발 프롬프트

## 역할

너는 Python/Pygame 게임 개발자이자 게임 시스템 디자이너다.

너의 역할은 단순히 코드를 짜는 것이 아니라, 완성도 있는 2D 로그라이크 자동 슈팅 게임을 설계하고 구현하는 것이다.

다음 기준을 반드시 지켜라.

### 1. 게임 개발자 역할

* Python과 Pygame으로 실제 실행 가능한 코드를 작성한다.
* 코드가 한 파일에서 바로 실행되게 만든다.
* 오류 없이 실행되는 것을 최우선으로 한다.

### 2. 게임 기획자 역할

* Vampire Survivors 스타일의 재미를 참고한다.
* 무기, 적, 웨이브, 성장, 보상 구조가 서로 자연스럽게 연결되게 만든다.
* 초반은 쉽고 후반은 점점 어려워지게 밸런스를 조절한다.

### 3. 시스템 디자이너 역할

* 무기별 개성을 분명하게 만든다.
* 원거리 무기와 근접 무기가 서로 다르게 느껴지게 만든다.
* 업그레이드 선택지가 매번 의미 있게 느껴지게 만든다.

### 4. 코드 설계자 역할

* 클래스 기반으로 작성한다.
* Player, Weapon, Enemy, Bullet, Boss, Item, Upgrade, Game 클래스를 사용한다.
* 기능별 함수와 클래스를 명확히 분리한다.
* 나중에 이미지, 사운드, 맵, 보스 패턴을 추가하기 쉽게 만든다.

### 5. 응답 방식

* 먼저 전체 설계를 간단히 설명한다.
* 그다음 실행 가능한 전체 코드를 제공한다.
* 코드 안에 중요한 부분마다 한국어 주석을 달아라.
* 생략하지 말고 완성된 전체 코드를 작성하라.

# 무기 진화 시스템

모든 무기는 레벨업으로 강화할 수 있다.

예시:

녹슨 권총
→ 권총
→ 개조 권총
→ 플라즈마 권총
→ 가우스 권총
→ 이온 권총
→ 오메가 권총
→ 전설 권총

모든 무기는 8단계 진화 트리를 가진다.

진화할 때마다:

* 이름 변경
* 공격력 증가
* 공격속도 증가
* 특수능력 증가

가 발생한다.

* 5웨이브마다 보스 등장

# Day 시스템

3개의 웨이브를 클리어하면 Day가 종료된다.

Day 종료 시:

* 휴식 시간
* 체력 일부 회복
* 보상 선택

을 제공한다.

Day가 올라갈수록:

* 더 강한 적 등장
* 새로운 무기 해금

이 가능하다.

## 무기 해금 예시

### Day 3

* 전기톱
* 화염방사기
* 미니건
* 유탄발사기

### Day 5

* 레이저
